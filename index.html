vuex-notes-app

A Notes App with Vuex

项目初始化

# 克隆项目代码
git clone https://github.com/lichenbuliren/vuex-notes-app.git

# 进入到项目根目录
cd vuex-notes-app

# 安装依赖包
npm install

# serve with hot reload at localhost:8080
npm run dev

常见问题

由于该例子使用的是sass写的样式，如果编译的时候，控制台出现ERROR in Cannot find module 'node-sass'，说明你的环境缺少全局的模块。

安装node-sass模块，推荐使用cnpm安装

cnpm install -g node-sass

*window用户请用管理员模式安装
